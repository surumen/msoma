Web Design Projects

Every detail in these projects is chosen carefully to ensure that you've mastered the content covered during all workshops for that week.
Projects are due Wednesday the following week. All projects must be done individually, except with the help of the M-Soma instructors and mentors. If you use code snippets that you did not write yourself, you must credit the author appropriately.
While we encourage collaboration, there are limits such as not showing or sharing your code to someone else in any form.

Project #1: Static Website

Project details
Due: Wednesday, June 14, 2017 at 11:59PM

In this project, you will build a website with multiple pages. You must have a home page, an about page, and a contact page. You must use a background image on each page, have a header, and a footer as well as links on the body section.

You must also have the following items in a list:

  • Unordered list
  • Ordered list
Total Points: 25 + 2 Extra Credit
