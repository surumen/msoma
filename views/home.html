Web Development

This course prepares you to build responsive and interactive websites without any prior knowledge. The first part of this course focuses on building Static websites using only HTML, CSS and JavaScript. The second part focuses on building Dynamic websites using Ruby on Rails on the back-end. You will be able to implement login/sign-up functionalities for your website and do more cool stuff.

Intro to Programming

This course introduces you to the fundamentals of programming. Our main goal is not to teach you a particular programming language, but rather to teach you programming as an art. We will start by introducing you to the tools you will need as a programmer, and then dive into software. After this course, you should be able to pick up any programming language fairly quickly.

Start Course

Design Thinking

This course introduces you to the mindset, skillset, and toolset associated with "human-centered design". If you ever dreamed of starting your own company, this is the place to start! From identifying problems, revealing new insights, and presenting possible solutions through sketching and prototypes, you will gain first-hand experience in this course.

Start Course

Social Implications of Computing

This is a discussion intensive course about the social implications of computing. The purpose of this course is to help you as a creator of technology to make informed and socially conscious decisions in your work. Each week there will be a discussion on technology topics of your choice, and a brief reading on the topics will be released before the workshops.
