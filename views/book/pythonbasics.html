2.1 Using Python

Running Python3 in the Terminal

Type python3 in your terminal. The >>> you see shows you that you're inside the Python Shell. Let's do some calculations using python. Do the following arithmetic. After typing the digits, hit Enter

>>> 5+5
                
10
                
>>> 100*7
                
700
                
>>> 3-5
                
-2

We can do more than just calculations with Python. Let's trying printing some statements

>>> print('Hello there')
                
Hello there
                
>>> print(20 + 5)
                
25

As you can see above, you can use Python to do all your calculations. There are many other mathematical expressions you can execute in the Python shell including division, modulo, among many others.

2.2 Data Types

In your primary school Mathematics class, you learned about Fractions, Decimals, Prime Numbers, and Algebra. All these can be referred to as Mathematical data types.

When it comes to Python, the data types are just as diverse. All these are things you already encountered in primary school and high school, the only difference is that now we are giving them their official names. Or the names used in computer programming. They are grouped into different classes. There are numeric data types, booleans, sequences, sets, and mappings.

Numeric Types

There are different numeric types. This applies to any programming language.

  • int: (Integers) these are just numbers like 1, 2, 3, 4, 100, 5000
  • float: these are decimal numbers like 3.14159265359
  • str: (String); these are expressions like "this is a string", "Monday". They are characters inside the quotation marks
  • bytes: a sequence of integers between 0 and 255.
  • list a list in python is a collection of data types. The list can contain same types or different types. Lists in python are enclosed inside parentheses. {3, 4, 6, 12, "Tuesday"}
  • tuple: a collection of values that can be treated as a unit. In many ways it's just like a list, but you cannot change the contents of a tuple whereas you can change the contents of a list.

a. Integers

Integers are whole numbers - from large negative numbers to large positive numbers. In a computer, the size of numbers we use is restricted although it's not restricted in real life. This is just due to the amount of memory in your computer - it can't handle numbers upto infinity. In fact, on most current computers and programming languages there is an upper limit on the size of numbers called MAXINT.

There are several arithmetic operations you can perform on integers. You can do addition, subtraction, multiplication, division, modulo, or exponentiation.


>>> print(10 + 20) # addition of 10 and 20
                
30
            

>>> print(40 - 1) # subtraction of 40 and 1
                
39
            

>>> print(4 * 7) # multiplication of 4 and 7
                
28
            

>>> print(80 / 10) # division of 80 and 10
                
8
            

>>> print(24 % 5) # modulo: the remainder of 24 divided by 5
                
4
            

>>> print(2**5) # exponentiation: 2 raised to power 5
                
32

b. Strings

In Python, strings can be represented in several ways.

  • With single quotes: 'This is a string with single quotes'
  • With double quotes: "This string has double quotes"

  • With triple quotes: """ This string has triple quotes. Sometimes you may want to use this if you have a very long string """

    • You can perform different operations on strings. You can join two strings, split a string to sub-strings, copy one string to another, etc. Some of these operations are built into the Python language so using them is just straightforward. Others are not built into Python - we will go into details on how to use some operations that are not built into a language. Let's look at two built in operations - combining two strings into one, and repeating a string several times.

    Combining two strings into one is called concatenation. Open your terminal and try concatenating these strings.

    >>> print('Again and ' + 'again') # string concatenation
                
    Again and again
            
    >>> print('Repeat ' * 3) # string repetition
                
    Repeat Repeat Repeat
            
    >>> print('Again ' + ('and again' * 3) # string concatenation and repetition
                
    Again and again and again and again

    Booleans

    A boolean is a true or false value. Sometimes on a computer boolean values are represented using 0 or 1. (0 for false, and 1 for true). Boolean values are sometimes known as "truth values" because they are used to test whether something is true or not.

    Sets

    A set is an unordered collection of unique objects. A frozen set is a set whose values cannot be changed.

    Mutable vs Immutable objects

    A mutable object is one whose contents can be changed, while an immutable object is one whose contents cannot be changed.

    2.3 Elements of Computer Programs

    Computer programs can be structured in four ways. When defining the term "algorithm" in Chapter 1, we said that it is a sequence of steps, it applies a process that specifies when each step is executed, and has a means of determining when to stop. When we are describing the common features of all programs, we will be just going into more details on this.

    These are the four main features we will explore:

    • Sequences of instructions

      This program follows these steps in strict sequence. It does not execute Step 3 before 2 and so on.
    • Branches

      When this program reaches a certain point, it makes the decision whether to take the upper branch or the lower branch. Sometimes you might also be faced with a dilemma, where you have to choose between two or even more situations. Computer programs are just written to solve real world problems so such situations can also occur. This is called branching.
    • Loops

      You eat food everytime you get hungry regardless of whether you only ate three hours ago, or you last ate the previous night. Eating is an endless loop in the human life. The test condition is "if or when you are hungry", then repeat the action which is eating. Computer programs are just like that. There are some operations that we need to repeat for many reasons. In that case we use loops. We will go into more details on this in Chapter 3.
    • Functions

      Sometimes you might have to repeat similar actions even when you're doing difference things. If for example you're making ugali, you need to clean your utensils first. If you're making uji, you also have to clean your utensils first. So if making ugali was Step 1 and making uji was Step 3, they both share some other action - which we're calling the shared module here. Computer programs also sometimes perform an identical sequence of actions several times. For convenience, these common actions are place in a module (also called function, or procedure). If you want to clean your utensils, you put all of them in the sink. That's what a module/function does here.

    Now we have seen that you can place common actions in one place called a function when designing computer programs. Let's now start designing our first functions.

    2.4 Defining Functions

    So far what we've done is just using Python to do arithmetic operations, and printing some statements. We also saw that the fourth element of computer programs is that some actions might be repeated many times. If an action is repeated we can place it in a function or module, and then everytime we need to use it, we use that function we created.

    Let's do some more printing before we define our first function. If you're celebrating your birthday this month, let's make Python sing a little song for you.

    >>> print('Happy Birthday to you!')
            
    >>> print('Happy Birthday to you!')
            
    >>> print('Happy Birthday, dear friend')
            
    >>> print('Happy Birthday to you!')

    If I am also celebrating my birthday, it's a lot of work to keep writing the same code over and over again. To avoid this, we can create modules or functions, for each person who is celebrating their birthday.

    Moses is celebrating his birthday, so let's create a function that makes Python "sing" the birthday song for him.

    In Python, we will need to give our function a name so that we don't confuse with other functions. We are going to call it happyBirthdayMoses. When creating a function, we say we are defining it. We use the def keyword, short for define.

                    def happyBirthdayMoses():
    
                    print("Happy Birthday to you!")
    
                    print("Happy Birthday to you!")
    
                    print("Happy Birthday, dear Moses!")
    
                    print("Happy Birthday to you!")

    Before we make this function print the song, it is important to first understand every line we just wrote.

    • Line 1: Contains def, the name of the function, parentheses, and a colon
      All Python functions are written like this. The standard syntax for defining a function in Python is
      def function_name():
    • Line 2-5: Are indented by 4 spaces.
      This is called a whitespace. If you fail to indent your code by this exact amount of spaces, you will get Syntax errors. That should not worry you, however, if you encounter this just correct the indentation on your code.
